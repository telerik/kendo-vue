# Kendo UI for Vue Documentation Demos

Runnable demos extracted from the Kendo UI for Vue documentation.

## Quick Start

```bash
npm install
npm run dev
```

Then open [http://localhost:3030](http://localhost:3030) to browse all demos.

## Requirements

- Node.js 20+
- A valid [Kendo UI for Vue license](https://www.telerik.com/kendo-vue-ui/pricing/)

## Structure

- `content/` — Documentation markdown, API docs, and demo source files (synced automatically)
- `main.ts` — App entry point with auto-discovery of all demos
- `vite.config.ts` — Vite configuration

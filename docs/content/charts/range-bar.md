# Range Bar

Range Bar charts are categorical charts which represent a variation of the Bar charts.

Range Bar charts display data as bars where each bar represents a value range that spans between its minimum and maximum levels. Unlike the standard Bar charts, which have anchored bars to their X axis, Range Bar charts render floating bars.

# AxisLabelsPosition
The position of the axis labels.

```jsx
 <Chart>
   <ChartValueAxis>
     <ChartValueAxisItem :labels="{ position: 'end' }"/>
   </ChartValueAxis>
 </Chart>
);

```


`type` AxisLabelsPosition = "start" | "end" | "onAxis" | "";

# AxisTitlePosition
The position of an axis title.

```jsx
 <Chart>
   <ChartValueAxis>
     <ChartValueAxisItem :title="{ text: 'Title', position: 'top' }"/>
   </ChartValueAxis>
 </Chart>

```


`type` AxisTitlePosition = "top" | "bottom" | "left" | "right" | "center";

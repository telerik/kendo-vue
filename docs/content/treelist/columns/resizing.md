# Resizing

The Kendo UI for Vue Native Data TreeList enables you to resize its columns by dragging the edges (resize handles) of the header cells.

To enable column resizing, set the [`resizable`]({% slug api_treelist_treelistprops %}#toc-resizable) property of the TreeList to `true`.

# Appearance

The `FormElement` component allows selecting a predefined size option that changes the spacing within the form.

## Size

The [`size`]({% slug api_form_formelementprops %}#toc-size) property controls the overall size of the form element and the spacing within the form. The available size options are:

- `small`
- `medium` (default)
- `large`
- `null`&mdash;Does not apply any size class.

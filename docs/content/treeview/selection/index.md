# Selection

A TreeView item is selected when its `selected` field is set to `true`.

Instead of `selected`, you can implement your own field by setting the [`selectField`]({% slug api_treeview_treeviewprops %}#toc-selectfield) property to the name of that custom field.

The TreeView supports both single and multiple selections. When the selection is `multiple`, set the [`aria-multiselectable`]({% slug api_treeview_treeviewprops %}#toc-ariamultiselectable) property to `true` for better [accessibility]({% slug accessibility_treeview %}).

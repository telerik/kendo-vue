# Kendo UI for Vue Native TreeView Overview

The Kendo UI for Vue Native TreeView displays hierarchical data in a traditional tree structure, supports user interaction through mouse or touch events, and performs reordering operations by using the drag-and-drop functionality.

The TreeView component is part of the Kendo UI for Vue Native library of Vue UI components. It is distributed through NPM under the [kendo-vue-treeview package](https://www.npmjs.com/package/@progress/kendo-vue-treeview).

## Key Features

The Kendo UI for Vue Native TreeView provides a wide and always growing range of built-in features that isn't limited by the list in this section. This developer-friendly feature set allows you to create beautiful and functional applications faster. At the same time, the Kendo UI for Vue Native development team works constantly to improve the performance of the components and their capabilities.

### Data Binding

The TreeView has features that enable you to work with various types of collections and help you visualize a hierarchical list. The components allow you to work with server data and load more items on-demand. [Read more about the data binding functionality of the TreeView...]({% slug databinding_treeview %})

### Dragging and Dropping

The TreeView provides features that enable you to drag and drop nodes inside the TreeView. [Read more about the drag and drop feature of the TreeView...]({% slug dragdrop_treeview %})

### Selection and Checkboxes

The TreeView provides features that allow you to select single or multiple nodes either by using node checkboxes or clicking the nodes themselves. Read more about the [selection feature]({% slug selection_treeview %}) and [checkbox feature of the TreeView...]({% slug checkboxes_treeview %})

### Custom Rendering

You can visually customize the content of the TreeView items by utilizing the custom item rendering. [Read more about the custom item rendering of the TreeView...]({% slug rendering_treeview %})

### Appearance

The TreeView provides a predefined set of appearance options. These alternative styling options enable you to configure each aspect of the appearance of a TreeView. [Read more about the appearance TreeView...]({% slug appearance_treeview %})

### Keyboard Navigation

The keyboard navigation of the TreeView provides a set of keyboard shortcuts that yon use to interact with the component. [Read more about the keyboard navigation of the TreeView...]({% slug keyboard_navigation_treeview %})

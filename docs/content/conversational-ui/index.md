# Kendo UI for Vue Native Conversational UI Overview

The Kendo UI for Vue Conversational UI package provides the [AIPrompt]({% slug overview_aiprompt %}) component that allows you to effortlessly integrate an interface for submitting and enhancing AI prompts to selected Large Language Models and organizing the generated output in predefined or tailor made lists or responses.

The Kendo UI for Vue AIPrompt component is distributed through the [kendo-vue-conversational-ui NPM package](https://www.npmjs.com/package/@progress/kendo-vue-conversational-ui).

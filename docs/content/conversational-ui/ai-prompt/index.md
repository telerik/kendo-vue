# Kendo UI for Vue AIPrompt Overview

The Kendo UI for Vue Native AIPrompt enables you to write prompts and execute predefined commands as well as look and interact with the output from the AI.

The Kendo UI for Vue AIPrompt component is distributed through the [kendo-vue-conversational-ui NPM package](https://www.npmjs.com/package/@progress/kendo-vue-conversational-ui).

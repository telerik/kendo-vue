# AIPrompt Commands

The Kendo UI for Vue AIPrompt allows you to define custom commands that can be used by the users to fine-tune a response. The custom commands are handled by the [`AIPromptCommandsView`]({% slug api_conversational-ui_aipromptcommandsviewprops %}) component.

When defined, the commands can be accessed using the `...` button in the AIPrompt's toolbar.

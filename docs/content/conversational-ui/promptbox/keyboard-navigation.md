# Keyboard Navigation

All PromptBox functions are accessible to keyboard users.

The PromptBox supports the following keyboard shortcuts:

| Shortcut          | Description                                         |
| :---------------- | :-------------------------------------------------- |
| `Enter`           | Submits the prompt when `loading` state is `false`. |
| `Shift` + `Enter` | Inserts a new line in multi-line and auto modes     |
| `Tab`             | Moves focus to the next focusable element           |
| `Shift` + `Tab`   | Moves focus to the previous focusable element       |

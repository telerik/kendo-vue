# Kendo UI for Vue Native ChunkProgressBar Overview

The ChunkProgressBar displays and tracks the progress of a task as a predefined number of chunks.

It supports horizontal and vertical orientation, and different directions.

>The ChunkProgressBar is part of the [Kendo UI for Vue Native ProgressBars]({% slug overview_progressbars %}) component library. The procedures for installing, importing, and using the ProgressBars are identical for all components in the package. To learn how to use the ChunkProgressBar and the rest of the ProgressBars, see the [Getting Started with the Kendo UI for Vue Native ProgressBars]({% slug get_started_progressbars %}) guide.

## Key Features

* [Value and ranges]({% slug chunkprogressbar_value_ranges %})&mdash;You can set the value and the value range of the ChunkProgressBar component.
* [Disabled ChunkProgressBar]({% slug chunkprogressbar_disabled %})&mdash;You can disable the Kendo UI for Vue Native ChunkProgressBar component.
* [Orientation]({% slug chunkprogressbar_orientation %})&mdash;You can set the orientation to `horizontal` or `vertical`.
* [Direction]({% slug chunkprogressbar_direction %})&mdash;You can reserve the direction of the ChunkProgressBar component.
* [Appearance]({% slug chunkprogressbar_appearance %})&mdash;You can quickly style the ChunkProgressBar by choosing from the predefined styling options.
* [Globalization]({% slug rtl_progressbars %})&mdash;The built-in globalization support allows you to create apps that are ready to be implemented worldwide.

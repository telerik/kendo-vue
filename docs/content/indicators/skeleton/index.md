# Kendo UI for Vue Skeleton Overview

The Native Skeleton component helps developers to speed up the initial page load time by displaying a simplified preview of the content. This preview acts as a placeholder for the content while it loads. At the same time, the user will get a general idea about what will appear on the page.

The Kendo UI for Vue Native Skeleton is distributed through the [kendo-vue-indicators NPM package](https://www.npmjs.com/package/@progress/kendo-vue-indicators).

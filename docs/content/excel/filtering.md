# Filtering

The `saveExcel` method enables you to filter the data in the columns of the exported Excel file.

To enable filtering, set to `true` the [`filterable`]({% slug api_excel-export_excelexportoptions %}#toc-filterable) property of the object passed to the `saveExcel` method.

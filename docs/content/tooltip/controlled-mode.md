# Controlled Mode

By default, the Tooltip is in an uncontrolled state and is displayed as the built-in browser Tooltip on `mouseover`.

The Kendo UI for Vue Native Tooltip provides options for you to show it programmatically by using the [`open`]({% slug api_tooltip_tooltipprops %}#toc-open) and the [`targetElement`]({% slug api_tooltip_tooltipprops %}#toc-targetElement) properties.

# Events

The Kendo UI for Vue Native PanelBar provides the below events that you can handle. This allows you to tailor the behavior of the PanelBar to your specific requirements.

## List of available PanelBar events
* [PanelBarOnSelectEvent]({% slug api_layout_panelbarselecteventarguments %}) &mdash;Triggered when an  item inside the `PanelBar` is selected.

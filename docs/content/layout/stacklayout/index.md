# Kendo UI for Vue Native StackLayout Overview

The Kendo UI for Vue Native StackLayout component allows you to effortlessly align vertically or horizontally multiple elements in a stack.

It delivers different orientations, alignments, spacing and other handy options.

<div data-component="StartFreeTrialSection"></div>

The following example demonstrates the StackLayout component in action and also showcases other Kendo UI for Vue Native components&mdash;the [ButtonGroup]({% slug overview_buttongroup %}), [Avatars]({% slug overview_avatar %}) and [Cards]({% slug overview_card %}) components. The components are arranged in a stack and can be switched from horizontal to vertical orientation.

To learn how to use the StackLayout and the rest of the components in the package, see the [Getting Started with the Kendo UI for Vue Native Layout Package]({% slug getstarted_layout %}) guide.

## Key Features

- [Orientation]({% slug layout_stacklayout_orientation %})&mdash;The orientation configuration option allows you to choose between the vertical and horizontal positioning.
- [Gaps]({% slug layout_stacklayout_gaps %})&mdash;The Gap property allows you to set gaps between the boxes.
- [Horizontal alignment]({% slug layout_stacklayout_alignment %}#toc-horizontal-alignment)&mdash;You can align the StackLayout horizontally.
- [Vertical alignment]({% slug layout_stacklayout_alignment %}#toc-vertical-alignment)&mdash;You can align the StackLayout vertically.
- [Nested StackLayouts]({% slug layout_stacklayout_nested %})&mdash;You can create complex layouts with nested boxes.

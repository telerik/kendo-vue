# Kendo UI for Vue Native Menu Overview

The Kendo UI for Vue Menu is a multi-level component for displaying hierarchical data.

You can describe the Menu items either by setting the [`items`]({% slug api_layout_menuprops %}#toc-items) property.

The Menu component is part of the Kendo UI for Vue library of Native Vue components. It is distributed through NPM under the [kendo-vue-layout package](https://www.npmjs.com/package/@progress/kendo-vue-layout).

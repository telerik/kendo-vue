# Events

This article provides details about the events available in the Stepper API. Using these events you can tailor the Stepper's behavior to your specific requirements.

<div data-component="StartFreeTrialSection"></div>

## List of available Stepper events
* [StepperChangeEvent]({% slug api_layout_stepperchangeevent %}) - Triggered when a Step in the Stepper is changed.
* [StepperFocusEvent]({% slug api_layout_stepperfocusevent %}) - Triggered when a Step in the Stepper is focused.

# Keyboard Navigation

The keyboard navigation of the Stepper is always available.

The Stepper component supports the following keyboard shortcuts.


| SHORTCUT                            | DESCRIPTION                                                                       |
|:---                                 |:---                                                                               |
| `Left Arrow`                        | Focuses previous Step.                                                            |
| `Up Arrow`                          | Focuses previous Step.                                                            |
| `Right Arrow`                       | Focuses next Step.                                                                |
| `Down Arrow`                        | Focuses next Step.                                                                |
| `Home`                              | Focuses the first Step                                                            |
| `End`                               | Focuses the last Step                                                             |
| `Enter`                             | Selects the focused Step.                                                         |
| `Space`                             | Selects the focused Step.                                                         |
| `Tab`                               | Blurs the Stepper and moves the focus to the next focusable element on page.      |
| `Shift+Tab`                         | Blurs the Stepper and moves the focus to the previous focusable element on page.  |

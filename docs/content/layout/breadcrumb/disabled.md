# Disabled Breadcrumb Component

The Breadcrumb component can be disabled by setting the [disabled]({% slug api_layout_breadcrumbprops %}#toc-disabled) property to `true`.

# Disabled Item

To disable Breadcrumb items: inside the [data]({% slug api_layout_breadcrumbprops %}#toc-data) object property of the desired items, set `disabled` to `true`.

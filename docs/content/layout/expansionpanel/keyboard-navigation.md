# Keyboard Navigation

The keyboard navigation of the ExpansionPanel is always available.

The ExpansionPanel supports the following keyboard shortcuts:

| SHORTCUT                  |DESCRIPTION
|:---                       |:---
| `Enter`&`Space`           |Expands and collapses the focused item.

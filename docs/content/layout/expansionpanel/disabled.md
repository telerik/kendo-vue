# Disabled State

The `ExpansionPanel` allows you to control if the user can or cannot interact with the component. By default, the ExpansionPanel is enabled and this behavior can be configured through the [`disabled`]({% slug api_layout_expansionpanelprops %}#toc-disabled) property.

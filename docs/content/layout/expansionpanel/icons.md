# Custom Icons

The ExpansionPanel provides the option to customize the icon that indicates the expanded/collapsed state of the component.

You can replace both the expand and collapse icons with icons by your choice by using the [`expandIcon`]({% slug api_layout_expansionpanelprops %}#toc-expandIcon) and [`collapseIcon`]({% slug api_layout_expansionpanelprops %}#toc-collapseIcon) properties.

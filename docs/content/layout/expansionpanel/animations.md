# Animations

The ExpansionPanel allows you to add expand and collapse animations to it by using the Kendo UI for Vue Native Animation components from the [`@progress/kendo-vue-animation`]({% slug overview_animation %}) package.

To animate the ExpansionPanel, wrap its content in the desired Kendo UI for Vue Native Animation component.

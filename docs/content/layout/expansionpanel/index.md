# Kendo UI for Vue Native ExpansionPanel Overview

The Kendo UI for Vue Native ExpansionPanel component allows you to display all types of information in summary or in detail. The component has two types of states - expanded and collapsed. By changing these states, the user can switch between the summary or the detailed view of the component.

To learn how to use the ExpansionPanel and the rest of the components in the package, see the [Getting Started with the Kendo UI for Vue Native Layout Package]({% slug getstarted_layout %}) guide.

## Key Features

* [Animations]({% slug animations_expansionpanel %})&mdash;The expand and collapse actions of the ExpansionPanel can be animated.
* [Icons]({% slug icons_expansionpanel %})&mdash;The ExpansionPanel enables you to customize its expand and collapse icon indicators.
* [Disabled ExpansionPanel]({% slug disabled_expansionpanel %})&mdash;You can use the configuration options of the ExpansionPanel to disable the whole group of buttons or a single button so that users are not able to interact with it.
* [Accessibility]({% slug accessibility_expansionpanel %})&mdash;The ExpansionPanel is accessible for screen readers and supports WAI-ARIA attributes.
* [Keyboard navigation]({% slug keyboard_navigation_expansionpanel %})&mdash;The ExpansionPanel supports a number of keyboard shortcuts for processing various commands.

# Action Buttons

The Card provides options for rendering custom action buttons and customizing their orientation and layout.

## Basic usage

To specify action buttons in the Card, you can use the `CardActions` component. You can arrange the content of the buttons either horizontally or vertically, by using the [`orientation`]({% slug api_layout_cardactionsprops %}#toc-orientation) option of the component. To specify their layout utilize the [`layout`]({% slug api_layout_cardactionsprops %}#toc-layout) property.

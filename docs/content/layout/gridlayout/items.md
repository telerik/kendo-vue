# GridLayout Items Arrangement

The GridLayout component enables you to control the arrangement and dimensions of its items.

You can arrange the layout of the items through the following configuration properties:

* [`col`]({% slug api_layout_gridlayoutprops %}#toc-cols)&mdash;Defines the column line from which the GridLayoutItem will start.
* [`row`]({% slug api_layout_gridlayoutprops %}#toc-rows)&mdash;Defines the row line from which the GridLayoutItem will start.
* [`colSpan`]({% slug api_layout_gridlayoutitemprops %}#toc-colspan)&mdash;Specifies the number of columns over which the GridLayoutItem will span.
* [`rowSpan`]({% slug api_layout_gridlayoutitemprops %}#toc-rowspan)&mdash;Specifies the number of rows over which the GridLayoutItem will span.

# Kendo UI for Vue Native GridLayout Overview

The Kendo UI for Vue Native GridLayout component allows you to effortlessly arrange the contents of the component in rows and columns in a grid structure.

It offers grid-layout system with rows and columns. It is based on the [CSS Grid Layout](https://css-tricks.com/snippets/css/complete-guide-grid/) where you can configure items per a specific row and column, as well as span content across them.

## Key Features

* [Rows and columns]({% slug layout_gridlayout_rowscolumns %})&mdash; To configure the appearance of the GridLayout, you can use the `rows` and `cols` properties.
* [Gaps]({% slug layout_gridlayout_gaps %})&mdash;The setting of the gaps is another way to alter the appearance of the GridLayout.
* [Horizontal alignment]({% slug layout_gridlayout_alignment %}#toc-horizontal-alignment)&mdash;Four predefined values allows you to control the horizontal alignment.
* [Vertical alignment]({% slug layout_gridlayout_alignment %}#toc-vertical-alignment)&mdash;Four predefined values allows you to control the vertical alignment.
* [Items Arrangement]({% slug items_gridlayout %})&mdash;The GridLayout enables you to control the arrangement of its items.

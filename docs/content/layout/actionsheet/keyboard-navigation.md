# Keyboard Navigation

By default, the keyboard navigation of the ActionSheet is enabled.

Below is the list with the keyboard shortcuts that the ActionSheet supports.

| SHORTCUT                            | DESCRIPTION                          |
|:---                                 |:---                                  |
| `Tab`                               | Focuses the next item                |
| `Shift + Tab`                       | Focuses the previous item            |
| `Enter`                             | Selects the focused item             |
| `Escape`                            | Closes the ActionSheet               |

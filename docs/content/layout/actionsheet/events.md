# ActionSheet Events

This article provides details about the events available in the Kendo UI for Vur Native ActionSheet API. Using the following events you can tailor the component's behavior to your specific requirements.

<div data-component="StartFreeTrialSection"></div>

## List of available ActionSheet events
* [onClose]({% slug api_layout_actionsheetprops %}#toc-onclose) - Triggered when the modal overlay of the ActionSheet is clicked.
* [onItemselect]({% slug api_layout_actionsheetprops %}#toc-onitemselect) - Triggered when an item in the ActionSheet is clicked or selected.

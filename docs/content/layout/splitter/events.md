# Splitter Events

The Kendo UI for Vue Native Splitter emits a number of events which enable you to control its behavior upon user interaction.

## List of available Splitter events
* [SplitterOnChangeEvent]({% slug api_layout_splitteronchangeevent %}) &mdash;Triggered when a pane inside the Splitter is resized, collapsed or expanded.

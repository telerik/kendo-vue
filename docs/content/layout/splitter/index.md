# Kendo UI for Vue Native Splitter Overview

The Kendo UI for Vue Native Splitter splits the page into sections and allows the user to control the page layout.

The Splitter component is part of the Kendo UI for Vue Native library of Vue components. It is distributed through NPM under the [kendo-vue-layout package](https://www.npmjs.com/package/@progress/kendo-vue-layout).

## Key Features

* [Orientation]({% slug orientation_splitter %})&mdash;You can arrange the Splitter panes either horizontally or vertically.
* [Pane settings]({% slug panes_splitter %})&mdash;You can use the Splitter options to configure the dimensions, resizing, collapsing, and scrolling of the panes.
* [Events]({% slug events_splitter %})&mdash;Check the events available in the Splitter.
* [Globalization]({% slug rtl_layout %})&mdash;The Splitter has built-in globalization support.
* [Keyboard navigation]({% slug keyboard_navigation_splitter %})&mdash;The Splitter supports various keyboard shortcuts.
* [Accessibility]({% slug accessibility_splitter %})&mdash;The Splitter is accessible by screen readers and provides full WAI-ARIA support.

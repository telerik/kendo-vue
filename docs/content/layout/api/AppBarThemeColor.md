# AppBarThemeColor
Specifies the theme color of the AppBar ([see example]({% slug appearance_appbar %})).

* The possible values are:
* `base` (Default)
* `primary`
* `secondary`
* `tertiary`
* `inverse`

`type` AppBarThemeColor = "base" | "primary" | "secondary" | "tertiary" | "inverse";

# AppBarPositionMode
Specifies the positionMode of the AppBar ([see example]({% slug positioning_appbar %}#toc-position-mode)).

* The possible values are:
* 'static' (Default)
* 'sticky'
* 'fixed'

`type` AppBarPositionMode = "static" | "sticky" | "fixed";

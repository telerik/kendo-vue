# StackLayoutOrientation
Specifies the orientation of the StackLayout ([see example]({% slug layout_stacklayout_orientation %})).

The possible values are:
* (Default)`horizontal`
* `vertical`

`type` StackLayoutOrientation = "horizontal" | "vertical";

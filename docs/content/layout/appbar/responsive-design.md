# Responsive Design

The responsive layouts adjust and adapt to any device screen size, whether it is a desktop, tablet, or a mobile phone.

The below example demonstrates how the AppBar can be configured to properly display its content on different screen sizes. The responsive behavior of the component is achieved by using [`CSS media queries`](https://developer.mozilla.org/en-US/docs/Web/CSS/Media_Queries/Using_media_queries). Resize the screen to observe the responsive layout.

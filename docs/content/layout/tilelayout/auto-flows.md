# Auto Flows

The TileLayout component allows you to control how the auto-placement algorithm works, specifying exactly how auto-placed items get flowed.
It is based on the [CSS grid-auto-flow](https://developer.mozilla.org/en-US/docs/Web/CSS/grid-auto-flow) property.

To set different auto flow algorithm, use the [`autoFlow`]({% slug api_layout_tilelayoutprops %}#toc-autoFlow) property.

# Events

This article provides details about the events available in the `ContextMenu` API. Using these events you can tailor the ContextMenu's behavior to your specific requirements.

## List of available ContextMenu events
* [MenuSelectEvent]({% slug api_layout_menuselectevent %}) - Triggered when an item in the ContextMenu is selected/clicked.

# Events

This article provides details about the different events available in the SplitButton API. Using these events you can tailor the SplitButton's behavior to your specific requirements.

## List of available SplitButton events
* [onButtonClick]({% slug api_buttons_splitbuttonclickevent %}) - Triggered when the Button inside the SplitButton is clicked.
* [onItemClick]({% slug api_buttons_splitbuttonitemclickevent %}) - Triggered when an item in the SplitButton popup is clicked.
* [onFocus]({% slug api_buttons_splitbuttonfocusevent %}) - Triggered when the SplitButton is focused. 
* [onBlur]({% slug api_buttons_splitbuttonblurevent %}) - Triggered when the SplitButton is blurred.
* [onOpen]({% slug api_buttons_splitbuttonopenevent %}) - Triggered when the popup of the SplitButton is opened. 
* [onClose]({% slug api_buttons_splitbuttoncloseevent %}) - Triggered when the popup of the SplitButton is closed.

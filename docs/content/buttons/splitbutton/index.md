# Kendo UI for Vue Native SplitButton Overview

The Kendo UI for Vue SplitButton allows the user to execute a default action that is bound to a Button or to choose a predefined action from a drop-down list.

The Kendo UI for Vue SplitButton is distributed through the [kendo-vue-buttons NPM package](https://www.npmjs.com/package/@progress/kendo-vue-buttons). See the [package overview]({% slug overview_buttons %}) for more details.

## Key Features

* [Disabled SplitButton]({% slug disabled_splitbutton %})&mdash;You can easily render a disabled SplitButton and prevent end-users from interacting with it.
* [Icon SplitButton]({% slug icons_splitbutton %})&mdash;The SplitButton allows you to display images, predefined or custom icons.
* [Popup and items customization]({% slug customization_splitbutton %})&mdash;You can customize the rendering and behavior of the popup and its items.
* [Data binding]({% slug binding_splitbutton %})&mdash;You can bind the SplitButton to various data types.
* [Appearance]({% slug appearance_splitbutton %})
* [Globalization]({% slug rtl_buttons %})&mdash;The built-in globalization support allows you to create apps that are ready to be implemented worldwide.
* [Keyboard navigation]({% slug keyboard_navigation_splitbutton %})&mdash;The SplitButton supports various keyboard shortcuts.
* [Accessibility]({% slug accessibility_splitbutton %})&mdash;The SplitButton is accessible for screen readers and supports WAI-ARIA attributes.

>To learn more about the appearance, anatomy, and accessibility of the SplitButton, visit the [Progress Design System documentation](https://www.telerik.com/design-system/docs/components/splitbutton/)—an information portal offering rich component usage guidelines, descriptions of the available style variables, and globalization support details.

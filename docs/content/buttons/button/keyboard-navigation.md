# Keyboard Navigation


## Managing the Focus


The Button is focusable by default, and follows the regular tab order of the page.

## Keyboard Navigation


The Button supports the following keyboard shortcuts:

| Shortcut | Behavior |
| -------- | -------- |
| `Enter` or `Space` | Triggers a click action on the Button. |

# Disabled Button

By default, the Kendo UI for Vue Native Button is enabled.

To disable it, set the [disabled]({% slug api_buttons_buttonprops %}#toc-disabled) property to `true`. By default, the Kendo Button remains focusable when disabled. To make it non-focusable, set the `tabindex` to `-1`.

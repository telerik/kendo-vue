# Primary Button

The Kendo UI for Vue Native Buttons enables you to highlight actions to the user by enhancing its appearance.

To achieve this behavior, set the [`themeColor`]({% slug api_buttons_buttoninterface %}#toc-themecolor) property to `primary`.

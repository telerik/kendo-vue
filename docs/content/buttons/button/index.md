# Kendo UI for Vue Native Button Overview

The Kendo UI for Vue Native Button provides a clickable UI functionality and enables you to display only textual content, or show predefined icons, images, and custom icons, and to render a combination of textual and image content.

The Kendo UI for Vue Button component is part of the Kendo UI for Vue library of Vue UI components. It is distributed through NPM under the [kendo-vue-buttons package](https://www.npmjs.com/package/@progress/kendo-vue-buttons).

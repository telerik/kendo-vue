# Tools

The Kendo UI for Vue Toolbar allows you to render Button, ButtonGroup, DropDownButton, SplitButton and custom button components in it.

To include components in the Toolbar, nest them directly inside the component's definition. You can separate the different Toolbar elements from one another using the [ToolbarSeparator]({% slug api_buttons_toolbarseparator %}) or [ToolbarSpacer]({% slug api_buttons_toolbarspacer %}).

# Toolbar Events

This article provides details about the different events available in the Toolbar API. Using these events you can tailor the Toolbar's behavior to your specific requirements.

## List of available Toolbar events
* [onToolbarResizeEvent]({% slug api_buttons_toolbarresizeevent %}) - Triggered when the Toolbar is resized.

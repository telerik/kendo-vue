# Kendo UI for Vue Native ChipList Overview

The ChipList allows you to maintain a set of selected chips.

The Kendo UI for Vue ChipList component is part of the Kendo UI for Vue library of Vue UI components. It is distributed through NPM under the [kendo-vue-buttons package](https://www.npmjs.com/package/@progress/kendo-vue-buttons).

# Appearance

Apart from the default vision of the ChipList, the [`size`]({% slug api_buttons_chiplistprops %}#toc-size) styling option enables you to configure the spacing between the Chips inside the ChipList.

The available options that can be passed to the `size` property are:
* `small` — Sets the `gap` to `4px`.
* `medium` — Sets the `gap` to `8px`.
* `large` — Sets the `gap` to `12px`.

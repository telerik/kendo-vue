# Kendo UI for Vue SegmentedControl Overview

The Kendo UI for Vue SegmentedControl displays a horizontal set of mutually exclusive buttons, allowing users to select a single option from a group. It is ideal for switching between views, modes, or filter states.

The Kendo UI for Vue SegmentedControl is distributed through the [kendo-vue-buttons NPM package](https://www.npmjs.com/package/@progress/kendo-vue-buttons).

## Key Features

-   [Controlled and Uncontrolled Modes](slug:controlled_segmentedcontrol)&mdash;The SegmentedControl supports both controlled mode (using the `value` prop) and uncontrolled mode (using the `defaultValue` prop).
-   [Appearance](slug:appearance_segmentedcontrol)&mdash;You can adjust the size and layout mode of the SegmentedControl.
-   [Customization](slug:customization_segmentedcontrol)&mdash;Use the `itemTemplate` prop to fully customize the appearance of each item.
-   [Disabled Items](slug:disabled_segmentedcontrol)&mdash;You can disable individual segments to prevent interaction.
-   [Keyboard Navigation](slug:keyboard_navigation_segmentedcontrol)&mdash;The SegmentedControl supports keyboard navigation for accessibility.

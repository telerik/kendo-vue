# Kendo UI for Vue Native ButtonGroup Overview

The Kendo UI for Vue Native ButtonGroup is a container for two or more Button components which can be separately configured to display only textual content, or show predefined icons, images, and custom icons, and to render a combination of textual and image content.

The Kendo UI for Vue ButtonGroup component is part of the Kendo UI for Vue library of Vue UI components. It is distributed through NPM under the [kendo-vue-buttons package](https://www.npmjs.com/package/@progress/kendo-vue-buttons).

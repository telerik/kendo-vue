# DropDownButton Events

This article provides details about the different events available in the DropDownButton API. Using these events you can tailor the DropDownButton's behavior to your specific requirements.

## List of available DropDownButton events
* [onItemClick]({% slug api_buttons_dropdownbuttonitemclickevent %}) - Triggered when an item in the DropDownButton is clicked.
* [onFocus]({% slug api_buttons_dropdownbuttonfocusevent %}) - Triggered when the DropDownButton is focused. 
* [onBlur]({% slug api_buttons_dropdownbuttonblurevent %}) - Triggered when the DropDownButton is blurred.
* [onOpen]({% slug api_buttons_dropdownbuttonopenevent %}) - Triggered when the popup of the DropDownButton is opened. 
* [onClose]({% slug api_buttons_dropdownbuttoncloseevent %}) - Triggered when the popup of the DropDownButton is closed.

# Kendo UI for Vue SmartPasteButton Overview

The Kendo UI for Vue SmartPasteButton is a specialized button component that leverages AI to intelligently fill form fields from clipboard content.

The Kendo UI for Vue SmartPasteButton is distributed through the [kendo-vue-buttons NPM package](https://www.npmjs.com/package/@progress/kendo-vue-buttons).

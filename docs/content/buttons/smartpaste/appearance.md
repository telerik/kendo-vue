# Appearance

The SmartPasteButton provides a set of predefined appearance options that allow you to customize its look and feel.

## Size

The `size` prop controls the overall size of the SmartPasteButton.

<demo metaUrl="buttons/smartpaste/size/" height="200"></demo>

## Border Radius

The `rounded` prop controls the border radius of the SmartPasteButton.

<demo metaUrl="buttons/smartpaste/rounded/" height="200"></demo>

## Fill Mode

The `fillMode` prop controls the fill style of the SmartPasteButton.

<demo metaUrl="buttons/smartpaste/fillmode/" height="200"></demo>

## Theme Colors

The `themeColor` prop controls the color scheme of the SmartPasteButton.

<demo metaUrl="buttons/smartpaste/themecolor/" height="200"></demo>

## Icons

You can add icons to the SmartPasteButton by using the `icon` or `svgIcon` prop.

<demo metaUrl="buttons/smartpaste/icons/" height="200"></demo>

# Keyboard Navigation

The SmartPasteButton supports the following keyboard shortcuts:

## Focus

The SmartPasteButton can be focused using the `Tab` key.

## Keyboard Shortcuts

| Shortcut  | Description                              |
| --------- | ---------------------------------------- |
| `Enter`   | Activates the SmartPasteButton.          |
| `Space`   | Activates the SmartPasteButton.          |
| `Tab`     | Moves focus to the next focusable element. |
| `Shift`+`Tab` | Moves focus to the previous focusable element. |

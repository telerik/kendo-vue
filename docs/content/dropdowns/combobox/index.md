# Kendo UI for Vue ComboBox Overview

The Kendo UI for Vue ComboBox is a form component that lets you choose a single predefined value from a list and is a richer version of the `<select>` element and supports filtering, virtualization, and entering of custom values.

The Kendo UI for Vue ComboBox component is part of the Kendo UI for Vue library of UI components. It is distributed through NPM under the [kendo-vue-dropdowns package](https://www.npmjs.com/package/@progress/kendo-vue-dropdowns).

# Keyboard Navigation

The keyboard navigation of the ComboBox is always available.

The ComboBox supports the following keyboard shortcuts:

| SHORTCUT                            | DESCRIPTION                          |
|:---                                 |:---                                  |
| `Up Arrow`                          | Selects the previous available item. |
| `Down Arrow`                        | Selects the next available item.     |
| `Enter`                             | Selects the highlighted item or submits a custom value. |
| `Esc`                               | Closes the popup.                    |
| `Alt`+`Down Arrow`                  | Opens the popup.                     |
| `Alt`+`Up Arrow`                    | Closes the popup.                    |

# Keyboard Navigation

The keyboard navigation of the MultiSelect is always available.

The MultiSelect supports the following keyboard shortcuts:

| SHORTCUT                            | DESCRIPTION                          |
|:---                                 |:---                                  |
| `Up Arrow`                          | Focuses the previous available item. |
| `Down Arrow`                        | Focuses the next available item.     |
| `Home`                              | Focuses the first selected tag.      |
| `End`                               | Focuses the last selected tag.       |
| `Enter`                             | Selects the highlighted item.        |
| `Esc`                               | Closes the popup.                    |
| `Alt`+`Down Arrow`                  | Opens the popup.                     |
| `Alt`+`Up Arrow`                    | Closes the popup.                    |

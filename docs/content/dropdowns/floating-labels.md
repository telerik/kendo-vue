# Floating Labels

The DropDowns are compatible with the [Floating Label]({% slug floating_label_labels %}) component.

The following example demonstrates how to configure the `Floating Label` to work with the `AutoComplete`, `ComboBox`, `DropDownList`, `MultiSelect`,
 `DropDownTree` and `MultiSelectTree` components. To add the `Floating Label`, define it by passing a value to the `label` property of the different date and time inputs.

# PDF Export Basics

The [`GridPDFExport`]({% slug api_pdf_gridpdfexportprops %}) component provides options for exporting the content of the native Vue Grid by Kendo UI to PDF.

## Getting Started

To enable the PDF export, wrap the Grid with the `GridPDFExport` component. To initiate the PDF export, use the `save` method.

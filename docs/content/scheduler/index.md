# Kendo UI for Vue Scheduler Overview

The Kendo UI for Vue Native Scheduler offers built-in support for timezone conversion, resource visualization, grouping, editing, and recurring events.

Each feature of Scheduler is supported by all its built-in views which includes:

* [**Timeline** view](slug:timeline_view_scheduler)
* [**Day** view](slug:day_view_scheduler)
* [**Week** view](slug:week_views_scheduler)
* [**Month** view](slug:month_view_scheduler)

## Vue Scheduler Features

Among the features, which the Kendo UI for Vue Native Scheduler delivers are:

* [Resources]({% slug resources_scheduler %})&mdash;You can visualize an event by coloring it depending on a field value.
* [Grouping]({% slug grouping_scheduler %})&mdash;The events of the Scheduler can be grouped based on their resources and field values.
* [Editing]({% slug editing_scheduler %})&mdash;The Scheduler enables the user to edit its events by dragging them to different slots, change their duration by resizing, and update any of their fields with the advanced edit form.
<!-- * [Recurring events]({% slug recurring_events_scheduler %})&mdash;Displaying occurrences of recurring events is based on the `recurrenceRule` and `recurrenceExceptions` properties of the component. -->
* [Timezones conversion]({% slug timezones_scheduler %})&mdash;The Scheduler can use the timezone of the client machine or a predefined timezone, and can also convert all events to the desired timezone.
* [Adaptive Slot Height]({% slug adaptive_slot_height_scheduler %})&mdash;The SchedulerViewSlot can dynamically change it's height to order to fit a predefined number of items.
* [Auto Item Height]({% slug auto_item_height_scheduler %})&mdash;The SchedulerViewItem can automatically resize depending on the size of its content, or accept a constant value.
* [Globalization]({% slug globalization_scheduler %})&mdash;By using the available globalization options in Kendo UI for Vue, you can translate the Scheduler messages by adapting them to specific culture locales.

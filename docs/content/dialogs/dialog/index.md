# Kendo UI for Vue Dialog Overview

The KendoVue Dialog communicates specific information and prompts users to take specific actions by interacting with a modal dialog.

The KendoVue Dialog component is part of the KendoVue library of Vue UI components. It is distributed through NPM under the [kendo-vue-dialogs package](https://www.npmjs.com/package/@progress/kendo-vue-dialogs).

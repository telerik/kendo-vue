# Show or hide the buttons inside the title of the Window

The visibility of the title bar commands can be controlled through the following properties:

* [closeButton]({% slug api_dialogs_windowprops %}#toc-closebutton) - Specifies if the Window will render a close button.
* [maximizeButton]({% slug api_dialogs_windowprops %}#toc-maximizebutton) - Specifies if the Window will render a maximize button.
* [minimizeButton]({% slug api_dialogs_windowprops %}#toc-minimizebutton) - Specifies if the Window will render a minimize button.

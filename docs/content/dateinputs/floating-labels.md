# Floating Labels

The Date Inputs are compatible with the [Floating Label]({% slug floating_label_labels %}) component.

The following example demonstrates how to configure the `Floating Label` to work with the `DateInput`, `DatePicker`, `DateTimePicker` and `TimePicker` components. To add the `Floating Label`, define it by passing a value to the `label` property of the different date and time inputs.

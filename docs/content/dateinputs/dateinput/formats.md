# Formats

You can control the format of the DateInput by using the [`format`]({% slug api_dateinputs_dateinputprops %}#toc-format) property, which accepts string parameters.

By default, the `format` property is set to `'d'`. For more information on the date and number formats Kendo UI for Vue supports, refer to the [kendo-intl](https://github.com/telerik/kendo-intl#date-formatting) GitHub repository.

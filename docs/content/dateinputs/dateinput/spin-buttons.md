# Spin Buttons

The spin buttons of the DateInput increase or decrease the currently selected date part.

By default, the spin buttons are disabled and the [`spinners`]({% slug api_dateinputs_dateinputprops %}#toc-spinners) property is set to `false`. To render the spin buttons, set `spinners` to `true`.

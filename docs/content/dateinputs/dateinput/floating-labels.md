# Floating Labels

The `DateInput` can be fully integrated with the `Floating Label` component no matter of the Kendo Theme that you choose. To add a `Floating Label` to the `DateInput` define the [`label`]({% slug api_dateinputs_dateinputprops %}#toc-label) property of the component.

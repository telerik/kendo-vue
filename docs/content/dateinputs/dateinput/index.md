# Kendo UI for Vue Native DateInput Overview

The Kendo UI for Vue Native DateInput represents an input field that recognizes and formats scheduling values such as dates.

The Kendo UI for Vue Native DateInput component is part of the Kendo UI for Vue library of Vue UI components. It is distributed through NPM under the [kendo-vue-dateinputs package](https://www.npmjs.com/package/@progress/kendo-vue-dateinputs).

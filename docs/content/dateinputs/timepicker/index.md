# Kendo UI for Vue Native TimePicker Overview

The Kendo UI for Vue Native TimePicker represents a time-list where the user can enter or pick time values.

The Kendo UI for Vue TimePicker component is part of the Kendo UI for Vue library of Vue UI components. It is distributed through NPM under the [kendo-vue-dateinputs package](https://www.npmjs.com/package/@progress/kendo-vue-dateinputs).

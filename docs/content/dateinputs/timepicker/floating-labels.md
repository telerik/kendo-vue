# Floating Labels

The `TimePicker` can be fully integrated with the `Floating Label` component no matter of the Kendo Theme that you choose. To add a `Floating Label` to the `TimePicker` define the [`label`]({% slug api_dateinputs_timepickerprops %}#toc-label) property of the component.

# Reverse Range Selection

By default, the Calendar swaps the `start` and `end` dates of the value if the `end` is before the `start`.

To enable the reverse range selection, set the [`allowReverse`]({% slug api_dateinputs_calendarprops %}#toc-allowreverse) property of the Calendar to `true`.

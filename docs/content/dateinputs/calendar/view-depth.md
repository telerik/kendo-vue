# View Selection Depth

The Calendar enables you to control the view depth to which the user can navigate.

To determine its navigation view depth, use the [`bottomView`]({% slug api_dateinputs_calendarprops %}#toc-bottomview) property. To limit the topmost view that user can reach, use the [`topView`]({% slug api_dateinputs_calendarprops %}#toc-topview) property.

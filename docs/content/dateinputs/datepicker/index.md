# Kendo UI for Vue Native DatePicker Overview

The Kendo UI for Vue DatePicker component offers a customizable interface for the user to enter and pick dates supporting different locales.

Its features cover everything from setting a default value and selecting a date range to form validation and keyboard shortcuts.

## Vue DatePicker Features Preview and Demo

Among the many features which the DatePicker delivers are:

* [Setting the default value]({% slug default_value_datepicker %})&mdash;Effortlessly configure the initial value that the DatePicker renders.
* [Form validation]({% slug forms_datepicker %})&mdash;Validate input values, set validation requirements (such as minimum and maximum date) and prevent the submission of forms which are in an invalid state.
* [Formats]({% slug formats_datepicker %}) and [placeholders]({% slug placeholders_datepicker %})&mdash;Control the date format and implement placeholders for its input field, including custom formats.
* [Controlled state]({% slug controlled_datepicker %})&mdash;Control the date value and popup state of the DatePicker.
* [Disabled state]({% slug disabled_datepicker %})&mdash;To disable input, change a single property.
* [JSON integration]({% slug jsonintegration_datepicker %})&mdash;You can also bind the DatePicker to dates which are serialized as strings.
* [Custom rendering]({% slug custom_rendering_datepicker %})&mdash;You can replace the DatePicker input field, calendar and popup child components with custom ones.
* [Date limits]({% slug dateranges_datepicker %})&mdash;Enable the user to pick a date within a predefined min and max date range.
* [Focused dates]({% slug dates_datepicker %})&mdash;Configure the initial date value the DatePicker displays.
* [Appearance]({% slug appearance_datepicker %})
* [Accessibility support]({% slug accessibility_datepicker %})&mdash;The DatePicker is compliant with WAI-ARIA and Section 508.

This demo implements some of the features that are available in the DatePicker.

* When the Vue DatePicker is initially loaded, you can **render a specific default value for that date**.
* You can **change the date format** and **render a week number column**.
* With the **globalization feature**, you can change the languages and cultures by using the **Locale** drop-down list.

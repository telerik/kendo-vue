# Getting Started with Kendo UI for Vue Native DatePicker

The Kendo UI for Vue Native DatePicker combines the Kendo UI for Vue DateInput and Calendar components and enables the user to enter or pick a date value.

The Kendo UI for Vue DatePicker component is part of the Kendo UI for Vue library of Vue UI components. It is distributed through NPM under the [kendo-vue-dateinputs package](https://www.npmjs.com/package/@progress/kendo-vue-dateinputs).

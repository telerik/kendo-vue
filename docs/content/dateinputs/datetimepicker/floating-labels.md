# Floating Labels

The `DateTimePicker` can be fully integrated with the `Floating Label` component no matter of the Kendo Theme that you choose. To add a `Floating Label` to the `DateTimePicker` define the [`label`]({% slug api_dateinputs_datetimepickerprops %}#toc-label) property of the component.

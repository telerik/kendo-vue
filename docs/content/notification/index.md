# Kendo UI for Vue Native Notification Overview

The Kendo UI for Vue Native Notification renders a brief message to the user with information about the status of an application process and enables you to animate its rendering and control its position.

The Kendo UI for Vue Native Notification component is part of the Kendo UI for Vue library of Vue components. It is distributed through NPM under the [kendo-vue-notification package](https://www.npmjs.com/package/@progress/kendo-vue-notification).

## Key Features

The Kendo UI for Vue Native Notification service delivers a range of handy and developer-friendly features whose number and further development are not limited by the list in this section. The Kendo UI for Vue Native team constantly invests efforts to improve the performance, add more value to the existing Notification library, and develop new features.

### Notification Types

The Notification features built-in types, which enable you to display different notification variants such as success or error. [Read more about the built-in Notification types...]({% slug types_notification %})

### Animations

The Notification allows you to use different built-in animation effects, which enable you to visualize the display and disappearance of the notifications. [Read more about the built-in animations of the Notification...]({% slug animations_notification %})

### Positioning

The Notification enables you to align it to the browser viewport and to position it to a specific container. [Read more about the positioning options of the Notification...]({% slug positioning_notification %})

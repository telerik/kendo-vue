# Animations

The Notification allows you to add opening and closing animations to it by using the Kendo UI for Vue Native Animation components from the [`@progress/kendo-vue-animation`]({% slug overview_animation %}) package.

To animate the Notification, wrap it in the desired Kendo UI for Vue Native Animation component. 

The different [Animation Types]({% slug animationtypes_animation %}) tou can use are:

* Fade
* Push
* Zoom
* Reveal
* Expand
* Slide

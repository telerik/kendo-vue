# Kendo UI for Vue Native RadioButton Overview


The Kendo UI for Vue RadioButton lets the user to choose only one from multiple options.

The RadioButton is designed to replace the `<input type="radio">` HTML5 tag and provides the Kendo UI for Vue specific styling.

<div data-component="StartFreeTrialSection"></div>

## Basic Usage

The RadioButton is typically used within a RadioGroup with the same `name` attribute. All regular [`input type="radio"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input/radio) HTML attributes are applicable.

The Kendo UI for Vue RadioButton component is part of the Kendo UI for Vue library of Vue UI components. It is distributed through NPM under the [kendo-vue-inputs package](https://www.npmjs.com/package/@progress/kendo-vue-inputs).

# Kendo UI for Vue Native ColorPalette Overview

The Kendo UI for Vue Native ColorPalette renders colors by using sets of predefined colors (color presets) or by implementing a custom color palette.

The ColorPalette component is part of the Kendo UI for Vue Native library of Vue UI components. It is distributed through NPM under the [kendo-vue-inputs package](https://www.npmjs.com/package/@progress/kendo-vue-inputs).

To learn how to use the ColorPalette and the rest of the components in the package, see the [Getting Started with the Kendo UI for Vue Native Inputs]({% slug getstarted_inputs %}) guide.

## Key Features

* [Disabled ColorPalette]({% slug disabled_colorpalette %})&mdash;You can render a disabled ColorPalette by setting a single property and keep it disabled until certain requirements are met.
* [Controlled mode]({% slug controlled_colorpalette %})&mdash;Take advantage of the built-in options that allow you to control the value of the ColorPalette.
* [Color presets]({% slug presets_colorpalette %})&mdash;The ColorPalette provides a set of ready-to-use color presets.
* [Custom ColorPalette]({% slug custom_colorpalette %})&mdash;You can customize the rendering of the ColorPalette by specifying the number of its columns and tile size.
* [Keyboard navigation]({% slug keyboard_navigation_colorpalette %})&mdash; The ColorPalette supports keyboard shortcuts that allow the users to interact with the component.
* [Events]({% slug events_colorpalette %})&mdash;To tailor the behavior of the ColorPalette to your specific requirements, handle the events that the control generates.

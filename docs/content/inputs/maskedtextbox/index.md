# Kendo UI for Vue Native MaskedTextBox Overview

The Kendo UI for Vue Native MaskedTextBox uses a mask to control the input of the user.

The Kendo UI for Vue MaskedTextBox component is part of the Kendo UI for Vue library of UI components. It is distributed through NPM under the [kendo-vue-inputs package](https://www.npmjs.com/package/@progress/kendo-vue-inputs).

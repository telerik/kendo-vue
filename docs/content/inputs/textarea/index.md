# Kendo UI for Vue Native TextArea Overview

The Kendo UI for Vue Native TextArea component offers a customizable interface for the user to submit value on multiple lines.

Its features cover everything from setting a default value and character counter to form validation.

This demo implements some of the features that are available in the TextArea:

* You can **resize** the TextArea.
* You can use the **character counter** to count and limit the input characters.
* You can use the TextArea with **form validation**.

## Key Features

* [Disabled TextArea](slug:disabled_textarea)&mdash;You can render a disabled TextArea component and keep it disabled until certain requirements are met.
* [Controlled mode](slug:controlled_textarea)&mdash;Take advantage of the built-in options that allow you to control the value of the TextArea.
* [Default state](slug:default_textarea)&mdash;The TextArea allows you to set its default value.
* [Prefix & Suffix Adornments](slug:textarea_prefix_suffix)
* [Sizing](slug:sizing_textarea)&mdash;You can configure the auto size option and the initial height of the TextArea.
* [Appearance](slug:appearance_textarea) - Configure the appearance of the TextArea.
* [Characters Counter](slug:counter_textarea)&mdash;The TextArea component is customizable and allows you to integrate it with a character counter.
* [Forms support](slug:forms_support_textarea)&mdash;You can integrate the TextArea with an [HTML5 form](https://developer.mozilla.org/en-US/docs/Learn/HTML/Forms), a [Kendo UI for Vue Native Form]({% slug overview_form %}), or a third-party Vue form library.
* [Globalization](slug:globalization_inputs)&mdash;The Kendo UI for Vue Native TextArea comes with globalization features that allow you to create applications that can be used all over the world.

>To learn more about the appearance, anatomy, and accessibility of the TextArea, visit the [Progress Design System documentation](https://www.telerik.com/design-system/docs/components/textarea/). This information portal offers rich component usage guidelines, descriptions of the available style variables, and globalization support details.

# ColorPickerView
Specifies the view which the ColorPicker will render in the popup
([see example]({% slug combinedview_colorpicker %})).

The possible values are:
* (Default) `palette`
* `gradient`
* `combo`

`type` ColorPickerView = "gradient" | "palette" | "combo";

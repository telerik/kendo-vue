# Events

This article provides details about the event available in the `ColorPicker` API.

## List of available ColorPicker events
* [ColorPickerActiveColorClick]({% slug api_inputs_colorpickeractivecolorclick %}) - Triggered when one click on the left side of the component component(the part that contains the color square).
* [ColorPickerBlurEvent]({% slug api_inputs_colorpickerblurevent %}) - Triggered when the ColorPicker is blurred.
* [ColorPickerChangeEvent]({% slug api_inputs_colorpickerchangeevent %}) - Triggered when the value of the component is changed.
* [ColorPickerFocusEvent]({% slug api_inputs_colorpickerfocusevent %}) - Triggered when the ColorPicker is focused.

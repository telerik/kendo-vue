# Kendo UI for Vue Native ColorPicker Overview

The ColorPicker enables the user to select and submit color values.

The ColorPicker is designed to replace the `<input type="color">` HTML5 tag which is not widely supported in browsers.

>The ColorPicker is part of the [Kendo UI for Vue Native Inputs]({% slug overview_inputs %}) component library. To learn how to use the ColorPicker and the rest of the components in the package, see the [Getting Started with the Kendo UI for Vue Native Inputs]({% slug getstarted_inputs %}) guide.

## Key Features

* [Disabled ColorPicker]({% slug disabled_colorpicker %})&mdash;You can render a disabled ColorPicker and keep it disabled until certain requirements are met.
* [Controlled mode]({% slug controlled_colorpicker %})&mdash;Take advantage of the built-in options that allow you to control the value of the ColorPicker.
* [Default value]({% slug default_value_colorpicker %})&mdash;You can set an initial value for the ColorPicker.
* [Views]({% slug combinedview_colorpicker %})&mdash;You can configure the ColorPicker to render a color gradient, a color palette, or a combination of both.
* [Custom ColorPicker]({% slug customizecolorpicker_colorpicker %})&mdash;The built-in appearance customization options allow you to adjust the palette and gradient popup to your specific requirements.
* [Color Contrast tool]({% slug colorcontrast_colorpicker %})&mdash;The ColorPicker has a built-in tool verifying the contrast of the selected color.
* [Appearance]({% slug appearance_colorpicker %})&mdash;You can customize the look of the ColorPicker through its built-in properties.
* [Keyboard navigation]({% slug keyboard_navigation_colorpicker %})&mdash;The ColorPicker supports numerous keyboard shortcuts that allow users to interact with the component.
* [Events]({% slug events_colorpicker %})&mdash;The ColorPicker has a number of built-in events available in its API.

# Icon

By default, the Rating item is set to visualize a 'star' icon.

To change the default Rating icon set it's [`svgIcon`]({% slug api_inputs_ratingprops %}#toc-svgicon) and the [`svgIconOutline`]({% slug api_inputs_ratingprops %}#toc-svgiconoutline) properties to the new wanted values.

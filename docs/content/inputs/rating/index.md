# Kendo UI for Vue Rating Overview

The Kendo UI for Vue Native Rating component enables the user to increase, decrease, and select predefined values by clicking it's icons along or using the arrow keys.

The Kendo UI for Vue Rating component is part of the Kendo UI for Vue library. It is distributed through NPM under the [kendo-vue-inputs NPM package](https://www.npmjs.com/package/@progress/kendo-vue-inputs).

## Key Features

* [Selection]({% slug selection_rating %})&mdash;The Rating allows you to configure it for continuous or single selection mode.
* [Precision]({% slug precision_rating %})&mdash;You can configure the Rating and allow it to render half-full items.
* [Label]({% slug label_rating %})&mdash;The Rating allows you to use a custom string as a label.
* [Icon]({% slug icon_rating %})&mdash;The Rating allows you to change the default icon.
* [Readonly]({% slug readonly_rating %})&mdash;You can override the default active state of the Rating.
* [Disabled]({% slug disabled_rating %})&mdash;Render a disabled Rating and keep it disabled until your requirements are fulfilled.
* [Controlled mode]({% slug controlled_rating %})&mdash;Take advantage of the built-in options that allow you to control the value of the Rating.
* [Forms Support]({% slug forms_support_rating %})&mdash;You can effortlessly integrate the Rating with an [HTML5 form](https://developer.mozilla.org/en-US/docs/Learn/HTML/Forms), a [`Kendo UI for Vue Form`]({% slug overview_form %}), or a third-party Vue form library.
* [Rating Item customization]({% slug item_customization_ratingitem %})&mdash;You can use the RatingItem component to customize the appearance of the Rating or to transform the RatingItem children.
* [Keyboard navigation]({% slug keyboard_navigation_rating %})&mdash;The Rating supports many keyboard shortcuts that allow users to interact with the component.
* [Accessibility]({% slug accessibility_rating %})&mdash;The Rating component is accessible for screen readers and provides full WAI-ARIA support.

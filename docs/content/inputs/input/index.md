# Kendo UI for Vue Native Input Overview

The Kendo UI for Vue Native Input enables you to [style](#toc-basic-usage) and implement [floating labels]({% slug floatinglabels_inputs %}) in `input` elements.

The Kendo UI for Vue Native Input component is part of the Kendo UI for Vue Native library of Vue UI components. It is distributed through NPM under the [kendo-vue-inputs package](https://www.npmjs.com/package/@progress/kendo-vue-inputs).

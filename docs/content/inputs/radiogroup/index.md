# Kendo UI for Vue Native RadioGroup Overview


The Kendo UI for Vue Native RadioGroup is a wrapper used to group two or more RadioButton components.

The Kendo UI for Vue RadioGroup component is part of the Kendo UI for Vue library of Vue UI components. It is distributed through NPM under the [kendo-vue-inputs package](https://www.npmjs.com/package/@progress/kendo-vue-inputs).

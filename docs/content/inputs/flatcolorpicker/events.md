# Events

This article provides details about the event available in the `FlatColorPicker` API.

## List of available FlatColorPicker events
* [FlatColorPickerOnChangeEvent]({% slug api_inputs_flatcolorpickerprops %}#toc-onchange) - Triggered when the `Apply` button is clicked.
* [FlatColorPickerViewChangeEvent]({% slug api_inputs_flatcolorpickerviewchangeevent %}) - Triggered when the View of the FlatColorPicker is changed.

# Keyboard Navigation

By default, the keyboard navigation of the NumericTextBox is enabled.

Below is the list with the keyboard shortcuts that the NumericTextBox supports.

| SHORTCUT                            | DESCRIPTION         |
|:---                                 |:---                 |
| `Up Arrow`                          | Increases the displayed numeric value with a predefined step. |
| `Down Arrow`                        | Decreases the displayed numeric value with a predefined step. |

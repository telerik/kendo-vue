# Predefined Steps

You can define the step of the NumericTextBox through the [`step`]({% slug api_inputs_numerictextboxprops %}#toc-step) property.

When `step` is set, you can increase or decrease the value of the component that adopts the predefined step by using the spin buttons or the `Up` and `Down` arrow keys.

# Spin Buttons

You can use the configuration options of the NumericTextBox to manage the settings of its spin buttons.

When enabled, the spin buttons increase or decrease the value with the predefined step. By default, the [`spinners`]({% slug api_inputs %}#toc-spinners) property is set to `true`. To hide the buttons, set the property to `false`.

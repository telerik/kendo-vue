## Character Counter

A common scenario for the `TextBox` component is displaying a character counter along the input element. This could be achieved by utilizing adornments to display the counter and the [maxlength]({% slug api_inputs_textboxprops %}#toc-maxlength) property, which will restrict the user input up to the specified length.

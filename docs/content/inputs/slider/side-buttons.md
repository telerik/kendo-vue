# Side Buttons

When enabled, the side buttons of the Slider increase or decrease the component value with the predefined step.

By default, the [`buttons`]({% slug api_inputs_sliderprops %}#toc-buttons) property is set to `false`. To display the buttons, set the property to `true`.

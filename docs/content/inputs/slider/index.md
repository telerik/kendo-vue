# Kendo UI for Vue Native Slider Overview

The Kendo UI for Vue Slider enables the user to increase, decrease, and select predefined values by dragging its handle along the track or by clicking the side arrow buttons.

The Kendo UI for Vue Slider component is part of the Kendo UI for Vue library of Vue UI components. It is distributed through NPM under the [kendo-vue-inputs NPM package](https://www.npmjs.com/package/@progress/kendo-vue-inputs).

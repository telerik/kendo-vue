# Labels

You can provide names for specific Slider values by defining labels along the track.

The labels are rendered by the [SliderLabel Component]({% slug api_inputs_sliderlabelprops %}). The [SliderLabel properties]({% slug api_inputs_sliderlabelprops %}) allow you to configure the content, position, title, and click handler of the Slider.

# Kendo UI for Vue Native ColorGradient Overview

The Kendo UI for Vue Native ColorGradient renders a gradient, a hue and an alpha slider, and inputs to manually enter the desired color.

The Kendo UI for Vue Native ColorGradient component is part of the Kendo UI for Vue library. It is distributed through NPM under the [kendo-vue-inputs package](https://www.npmjs.com/package/@progress/kendo-vue-inputs).

To learn how to use the ColorGradient and the rest of the components in the package, see the [Getting Started with the Kendo UI for Vue Native Inputs]({% slug getstarted_inputs %}) guide.

## Key Features

* [Disabled ColorGradient]({% slug disabled_colorgradient %})&mdash;You can render a disabled ColorGradient by setting a single property and keep it disabled until certain requirements are met.
* [Controlled mode]({% slug controlled_colorgradient %})&mdash;You can take advantage of the built-in options that allow you to control the value of the ColorGradient.
* [Removing opacity inputs]({% slug no_opacity_colorgradient %})&mdash;The ColorGradient supports options that allow the user to specify whether an alpha slider and an alpha input will be rendered.

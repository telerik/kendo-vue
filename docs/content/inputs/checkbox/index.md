# Kendo UI for Vue Native Checkbox Overview


The Kendo UI for Vue Checkbox lets the user toggle between checked, unchecked and the optional indeterminate states.

The Checkbox is designed to replace the `<input type="checkbox">` HTML5 tag and provides the Kendo UI for Vue specific styling.

<div data-component="StartFreeTrialSection"></div>

## Basic Usage

The CheckBox is used to represent boolean values via a binary checked state. All regular [`input type="checkbox"`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input/checkbox) HTML attributes are applicable.

The Kendo UI for Vue Checkbox component is part of the Kendo UI for Vue library of Vue UI components. It is distributed through NPM under the [kendo-vue-inputs package](https://www.npmjs.com/package/@progress/kendo-vue-inputs).

# Kendo UI for Vue Native Icons(FontIcons) List

The Kendo UI for Vue Native suite delivers more than 500 integrated font icons for its components.

Icon fonts are fonts that contain vector glyphs instead of letters and numbers. You can style them:
- with CSS by using all styling properties that can be applied to regular text in a modern browser.
- by choosing from the predefined styling options described in the [Appearance](slug:appearance_icon) article.

You can also replace built-in component icons with other predefined icons provided by the Kendo UI for Vue suite.

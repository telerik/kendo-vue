# Kendo UI for Vue Native SVGIcon Overview

The Kendo UI for Vue Native SVGIcon is a component for displaying SVG icons and supports 500+ SVG icons delivered by the Kendo UI for Vue suite.

> Kendo UI Icons version `^5.x.x` requires Kendo UI for Vue `v9.0.0` or later. For more information, refer to the [Icon Package Compatibility](slug:icons#icon-package-compatibility) section.

## Key Features

The SVGIcon provides support for a number of [styling options]({% slug appearance_svgicon %}) for setting its color, size, and rendering behavior, and also a predefined list of SVG icons.

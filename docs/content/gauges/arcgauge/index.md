# Kendo UI for Vue ArcGauge Overview

The Kendo UI for Vue ArcGauge represents value ranges on the arc.

Compared to the [Radial Gauge]({% slug overview_radialgauge_gauges %}), this gauge uses a colored arc instead of an arrow pointer. This leaves the center free for a user-defined [Center Template]({% slug centertemplate_arcgauge %}).

The Kendo UI for Vue Native ArcGauge component is part of the Kendo UI for Vue library of Vue UI components. It is distributed through NPM under the [kendo-vue-gauges package](https://www.npmjs.com/package/@progress/kendo-vue-gauges).

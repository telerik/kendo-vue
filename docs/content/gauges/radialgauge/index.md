# Kendo UI for Vue RadialGauge Overview

The Kendo UI for Vue RadialGauge represents values on a circular arc.

The Kendo UI for Vue RadialGauge component is part of the Kendo UI for Vue Native library of Vue UI components. It is distributed through NPM under the [kendo-vue-gauges package](https://www.npmjs.com/package/@progress/kendo-vue-gauges).

# Hint Overview

The Hint component enables you to display a hint message to Vue components.

## Basic Usage

Add the Hint component after the desired editor. Then set the [`id`]({% slug api_labels_hintprops %}#toc-id) property of the Hint and the `aria-describedby` property of the editor.

## Text Direction

The Hint component allows changing the text direction between the start and the end of the element. To change the direction set the [`direction`]({% slug api_labels_hintprops %}#toc-id) property to `end`.

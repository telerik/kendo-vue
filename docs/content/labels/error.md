# Error Overview

The Error component enables you to display an error message to Vue components.

## Basic Usage

Add the Error component after the desired editor. Then set the [`id`]({% slug api_labels_errorprops %}#toc-id) property of the Error and the `aria-describedby` property of the editor.

## Text Direction

The Error component allows changing its text direction. To change the direction set the [`direction`]({% slug api_labels_errorprops %}#toc-id) property to `end`.

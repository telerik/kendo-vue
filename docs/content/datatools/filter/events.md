# Events

This article provides details about the events available in the Filter API. Using these events you can tailor the Filter's behavior to your specific requirements.

<div data-component="StartFreeTrialSection"></div>

## List of available Filter events
* [FilterChangeEvent]({% slug api_data-tools_filterchangeevent %}) - Triggered when a filter option is changed or when an expression or group is added or removed.

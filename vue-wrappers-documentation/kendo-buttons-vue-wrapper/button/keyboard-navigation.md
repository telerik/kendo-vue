# Keyboard Navigation

The keyboard navigation of the Button is always available.

The Button supports the following keyboard shortcuts:

| SHORTCUT                     |  DESCRIPTION                    |
|:---                          |:---                             |
| `Enter`                      | Triggers the `click` event.     |
| `Space`                      | Triggers the `click` event.     |

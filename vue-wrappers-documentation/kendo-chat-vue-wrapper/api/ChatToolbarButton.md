# Chat

## Directive

`kendo-chat-toolbar-button`

## Parent Components

* [`kendo-chat`]({% slug api_chatcomponent %})

## Props

### name `String`

Defines the name of the button.

### text `String`

Defines the text that will be rendered in the button.

### icon-class `String`

Defines the icon classes of the span that is rendered in the button.

### attr `Object`

Defines an object that is applied to the `button` element as an attribute.

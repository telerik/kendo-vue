# CurrentTimeMarkerProps

### current-time-marker-update-interval `Number`

The update interval (in milliseconds) of the `current time` marker.

### current-time-marker-use-local-timezone `Boolean`

If `current-time-marker-use-local-timezone` is set to `false`, the `current time` marker will be displayed by using the timezone of the Scheduler.
